# OKF Skills

This repository contains custom agent skills for working with the **Open Knowledge Format (OKF)**. OKF is a structured layout for documenting codebase concepts, architecture, and learnings.

## Included Skills

- **[okf](file:///D:/projects/okf-skills/okf/SKILL.md)**: Author and maintain Open Knowledge Format (OKF) bundles — directories of typed markdown concept files.
- **[okf-maintain](file:///D:/projects/okf-skills/okf-maintain/SKILL.md)**: Proactively update the OKF knowledge bundle after codebase edits to keep details fresh and accurate.
- **[okf-visualize](file:///D:/projects/okf-skills/okf-visualize/SKILL.md)**: Generate and display interactive, browser-based Cytoscape.js graph visualizations of your OKF bundles.
